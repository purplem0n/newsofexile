# NewsOfExile
A website that aims to properly arrange all POE1 and POE2 news and patch notes in a single place. Whenever I browse path of exile website for either poe1 or poe2 to check for latest news or patches, it's very confusing because the POE1 news and patch content is under `Home -> (News or Patch Notes)` while content for POE2 is under `Forum ->  Index -> Early Access Announcements` for poe2 news and `Forum -> Index -> Early Access Patch Notes` for poe2 patch notes. This pattern genuinely confused me specially when I was new to the game. You could just bookmarked them but you'd have to bookmark 4 different index page (`poe1 news`, `poe1 patch`, `poe2 news`, `poe2 patch`) and to me this looked very convoluted. So what if you only have 1 index page that already contains all news and patch pages of Path Of Exile 1/2 in one place with interactive filters? This is the website.
